Tic-Tac-Toe

Language

Source Code

This was a group project and required us to work collaborativley using GitHub's commit, push, merge and version control features. Java/Kotlin were used to write the backend of the game with Android Studio's frontend UI used to create the display.

The app includes a range of extra and personalised features including:

• Settings including pvp or pve, board size and win condition options.
• Player creation, where players can choose custom names, markers and avatars.
• Stats page to keep track of all wins, losses, draws, games played and win %.
